# ShopEZ - Full Stack E-commerce Application

## Overview
ShopEZ is a full-stack e-commerce web application that enables users to browse products, add items to cart, and place orders. The application simulates a real-world online shopping experience.

## Features
- User registration and login with secure password hashing (bcrypt)
- Product listing and category-based browsing
- Shopping cart functionality
- Order placement and management
- Admin panel for managing products and orders

## Tech Stack
- Frontend: React, HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB
